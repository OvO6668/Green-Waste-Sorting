项目简介：

本项目旨在解决小区里垃圾分类"最后一百米"的问题。居民扔垃圾经常扔错，社区雇人督导成本又高，有害垃圾混在普通垃圾里还存在安全隐患。因此我们设计了一台桌面智能垃圾分类原型机，能自动识别并分拣四类垃圾，减少社区污染与潜在危险。

设备采用"主从双层控制"架构：上位机使用RDK X5开发板负责图像识别与决策，下位机使用ESP32主控板驱动机械臂执行分拣动作。结构件通过 SolidWorks 设计、3D 打印制作，整机模块化拼装，线束隐藏、主板封闭，便于维护与迭代。

功能特性
四分类识别：可回收物、厨余垃圾、有害垃圾、其他垃圾
视觉识别：基于 YOLOv11 轻量化模型，识别 几 类常见垃圾
橘子、青苹果、废弃药盒、空白透明药丸板、化妆品空瓶、一次性纸杯、海绵
自动分拣：机械臂精准抓取并投放至对应分类仓
离线运行：无需联网、无需电脑，插电即用

硬件组成

| 组件 | 型号/规格 |
|------|----------|
| 上位机 | RDK X5（旭日 X3 芯片，10 TOPS 算力） |
| 摄像头 | 800 万像素 AF 自动对焦 USB2.0 免驱动 IMX179 |
| 下位机 | ESP32 主控板 |
| 机械臂 | 中型机械臂（35kg.cm + 20kg.cm 数字舵机，最大负载 500g） |
| 结构件 | SolidWorks 设计 + 3D 打印 |

工作流程

1. 开机自检：RDK X5 初始化，机械臂回原点
2. 拍照识别：摄像头拍摄分拣平台上的垃圾，YOLOv11 模型提取特征并分类
3. 容错处理：连续 3 秒未识别成功则重新识别，连续两次失败投至"其他垃圾"
4. 机械臂分拣：上位机通过 UART 串口向下位机发送指令，机械臂移动、抓取、精准投放至对应分类桶
5. 复位等待：机械臂回原点，等待下一次投放

技术栈

边缘计算：RDK X5 + 旭日 X3 芯片
目标检测：YOLOv11 轻量化模型
机械控制：ESP32 + 串口通信
结构设计：SolidWorks + 3D 打印
开发语言：Arduino C++	基础语法、舵机控制、串口通信	 Python	基础语法、OpenCV 图像处理、串口通信
